# Project Resume Builder

A web application that allows users to create, customize, and export professional resumes easily.

## Features
- Customizable sections (Education, Experience, Skills, etc.)
- Export to PDF
- Real-time preview of resumes

## Technologies Used
- HTML
- CSS
- JavaScript
